## Summary
适合Mind+Python模式的邮件发送库，推荐使用QQ邮件
在邮箱的**设置>账户**中开启**IMAP/SMTP服务**，然后**生成授权码**填到积木中。

参考教程：https://mc.dfrobot.com.cn/thread-313396-1-1.html
